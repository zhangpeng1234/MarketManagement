# MarketManagement
商超购物管理系统具有商品管理、前台收银、商品库存等功能，本项目使用java作为开发语言，java控制台显示界面，mysql数据库存储数据，可以有效地锻炼和运用java、mysql数据库及基本sql编程开发的能力。
